# Hatstack.nl website (in development)

![Site preview](./docs/screenshots/hatstack-example.png)


## Prerequisites
To run this project, you will need the following:
- node
- npm
- hexo-cli

# Install
Install dependencies via npm
```
npm install
```

# Usage

Serve the website using hexo:
```
hexo server
```
